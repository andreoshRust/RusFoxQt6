# RusFoxQt6

**Русскоязычный язык сценариев для автоматизации на базе Qt6**

RusFoxQt6 — это мощный русскоязычный язык программирования для создания сценариев автоматизации, вдохновлённый синтаксисом 1С и построенный на технологиях Qt6 и JavaScript (QJSEngine). Язык позволяет быстро создавать скрипты для работы с файлами, базами данных, сетевыми запросами, графикой, печатью и многим другим.
